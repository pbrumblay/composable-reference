# Contributing to Composable Reference

Thank you for your interest in contributing! To keep the project sustainable and legally protected, we require all contributors to follow these steps.

## 1. Developer Certificate of Origin (DCO)
We use the DCO to ensure that contributors have the right to submit their code. To "sign" your contribution, simply add a line to your git commit message:

`Signed-off-by: Your Name <your.email@example.com>`

*Tip: You can do this automatically by using `git commit -s`.*

## 2. Contributor Grant (CLA-Lite)
By submitting a Pull Request to this project, you agree to the following:
"I grant the project owner a non-exclusive, perpetual, irrevocable, royalty-free, worldwide license to use, reproduce, modify, and sublicense my contribution (including the right to release it under different license terms) as part of this project."

## 3. Process
1. Fork the repo.
2. Create your feature branch.
3. Commit your changes with the `-s` (sign-off) flag.
4. Submit a Pull Request.
